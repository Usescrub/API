## Before you contact us

Include these details for the fastest resolution:

1. **`request_id`** from the API response or webhook payload
2. **Endpoint** and HTTP method (e.g. `POST /fraud/analyze/`)
3. **Timestamp** (UTC) of the request
4. **Environment** (sandbox or production)
