This document outlines how to call Scrub's Fraud detection API.
Scrub is your trusted gateway to real-time identity verification, transaction analysis, fraud reporting, and account risk scoring, tailored to the Nigerian financial ecosystem.

Our API enables financial institutions, fintechs, and payment platforms to detect and prevent fraudulent activities with reliable, fast, and secure endpoints.

### Base URL:

```
https://odoopi.usescrub.io
```

### Authentication:

Currently open for testing. In production, authentication will be via API key passed in the `Authorization` header:

```http
Authorization: Bearer YOUR_API_KEY
```
