Scrub is your gateway to real-time identity verification, transaction fraud analysis, credit reporting, and account risk scoring — built for the Nigerian financial ecosystem.

Our API helps fintechs, banks, and payment platforms detect and prevent fraud with reliable, fast, and secure endpoints.

<FeatureCards />

## API host

All requests are JSON over HTTPS. Scrub serves two environments — items cannot be moved between them.

<ApiHostSwitcher />

## Authentication

Include your API key or JWT access token in every request:

```http
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

Every response includes a `request_id`. Include it when contacting support — similar to [Plaid's request tracing](https://plaid.com/docs/api/).

## Rate limits

| Tier | Limit |
|------|-------|
| Sandbox | 100 requests/hour |
| Production (Free) | 50 requests/hour |
| Enterprise | 10,000 requests/day |

Response headers: `X-RateLimit-Limit`, `X-RateLimit-Remaining`, `X-RateLimit-Reset`.

## Use cases

- **Fintechs** — KYC during onboarding, loan underwriting with credit scores
- **Banks** — Real-time transaction screening and watchlist checks
- **E-commerce** — Device fingerprinting and velocity checks at checkout
- **Regulators** — Network-wide fraud reporting and pattern matching
